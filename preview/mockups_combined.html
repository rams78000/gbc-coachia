Architecture de l'application

L'application est construite selon une architecture feature-first avec des modules bien séparés pour chaque fonctionnalité.

lib/
├── app.dart                   # Point d'entrée de l'application
├── config/                    # Configuration globale
│   ├── di/                    # Injection de dépendances
│   ├── router/                # Configuration des routes
│   └── theme/                 # Thèmes de l'application
├── features/                  # Modules fonctionnels
│   ├── dashboard/             # Tableau de bord
│   ├── chatbot/               # Assistant IA
│   ├── planner/               # Planification
│   ├── finance/               # Gestion financière
│   ├── documents/             # Génération de documents
│   ├── storage/               # Stockage de fichiers
│   ├── settings/              # Paramètres
│   └── onboarding/            # Tutoriel d'introduction
│       ├── data/              # Couche de données
│       ├── domain/            # Logique métier
│       └── presentation/      # Interface utilisateur
│           ├── bloc/          # Gestion d'état
│           ├── pages/         # Écrans
│           └── widgets/       # Composants UI
└── main.dart                  # Point d'entrée
